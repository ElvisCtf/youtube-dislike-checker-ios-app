## Tech Stack
- MVVM
- UIKit
- URLSession

## Requirements
- iOS version: 18.0
- Xcode version: 16.0
- Swift version: 5

## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/ElvisCtf/weather-ios-app.git
    ```
2. Build and run
